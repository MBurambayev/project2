## Списки

Чтобы добавить ненумированные списки, нужно пункты выделить звездочкой(*) например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, нужно пронумеровать:
1. Элемент 1
2. Элемент 2
